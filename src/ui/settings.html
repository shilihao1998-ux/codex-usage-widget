Refresh

Quota data comes from account/rateLimits/read on the local Codex app-server — the same call the official Codex app makes. Reading it costs no quota, and a push from the server refreshes the card immediately whenever a turn consumes quota.

History

Files

state.json holds the last snapshot — including your account email. history.jsonl records how your quota changed over time. Nothing is ever uploaded.

Updates

This is the app's only outbound request: an unauthenticated GET to the GitHub releases API. It sends no identifiers and installs nothing — it just tells you a new version exists. It matters because this widget reads an undocumented Codex API: if OpenAI changes it, an old build could show wrong numbers.
